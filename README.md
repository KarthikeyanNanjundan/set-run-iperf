# set-run-iperf
This script automates the iperf throughput test. It sets iperf on devices &amp; runs iperf throughput. Both TCP and UDP protocols test. Includes reverse throughput with both Rx &amp; Tx runs. Script automatically opens up multiple terminals with gnome(ubuntu) function and runs the live throughput test. Gathers the results and closes the terminals. Prints results of the test.
